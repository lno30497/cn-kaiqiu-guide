# cn-kaiqiu-guide

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为 HTML 资源的集中存放与版本管理。

## 项目简介

`cn-kaiqiu-guide` 旨在收集、整理并共享一些独立制作的 HTML 页面。这些页面可能包含各类信息、工具或参考内容，以静态 HTML 的形式呈现，便于直接打开和浏览。仓库中的页面彼此独立，更新和维护也各自进行。

## 目录说明

```
/
├── pages/          # 存放所有独立的 HTML 页面文件夹
│   ├── example-page/   # 示例页面目录（实际目录名根据页面内容命名）
│   └── ...
├── assets/         # 公共资源（如样式、脚本、图片等），非必选
└── README.md       # 本文件
```

- `pages/`：核心目录，每个子文件夹对应一个完整的 HTML 页面及其相关资源。
- `assets/`：若页面间存在可复用的公共资源，可统一存放于此。

## 页面归档说明

- 每个 HTML 页面独立存放在 `pages/` 下对应的子文件夹中。
- 页面文件命名应简洁明了，尽量反映页面内容或用途。
- 页面内不应包含指向具体商业、营销或推广性质的链接。
- 所有页面均为静态 HTML，不依赖后端服务，可直接在浏览器中打开。

## 维护说明

- 本仓库欢迎提交新页面或改进现有页面。
- 提交新页面时，请在 `pages/` 下新建文件夹，并将所有相关文件放置其中。
- 请确保页面内容不违反相关法律法规，不包含恶意代码或不良信息。
- 对于已归档的页面，可能会不定期进行更新或清理。

## 使用方式

1. 克隆本仓库到本地。
2. 进入 `pages/` 目录，找到感兴趣的页面文件夹。
3. 直接使用浏览器打开其中的 `.html` 文件即可查看。
